# Uganda Extension Manuals

Place Ugandan extension manuals here as `.txt` or `.md` files.
The retrieval service will chunk each file by paragraph and score relevance.

Recommended:
- Use plain text exports of official manuals.
- One topic per file when possible (e.g., maize.txt, beans.txt).
- For language-specific text, suffix files with `_lg` or `_nyn` (e.g., `maize_lg.txt`).
- Optional structured JSON: `manuals.json`, `manuals_lg.json`, `manuals_nyn.json`.
- Optional pest cards: `pest_cards.json`, `pest_cards_lg.json`, `pest_cards_nyn.json`.
